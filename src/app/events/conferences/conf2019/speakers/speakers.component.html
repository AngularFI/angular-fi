Stanimira Vlaeva

Stanimira is a software engineer at Progress and a core team member for NativeScript, focused on the integration with Angular and performance optimizations. Her current work includes providing an official way for sharing code between web and mobile Angular with NativeScript and schematics, building a custom webpack build pipeline, and optimizing app launch performance. She loves spreading the word about NativeScript. Some of the recent events she's given a talk at are ngAtlanta, ngChina, AngularConnect, NativeScript Developer Days and DevReach.
