From web to mobile with Angular and NativeScript

Wednesday 13th March, 10.00-17.00
Expanding your skillset

We will start with a 30-40 min introduction to NativeScript - how it works and what it does.

Then, to get used to the NativeScript UI and layouts, we will build a really simple NativeScript app at http://play.nativescript.org (no local setup required).

For the rest of the workshop, we will take an existing Angular web application and add a mobile app to it. We will use the Angular CLI, the NativeScript schematics collection, and our bare hands to build it!

Required setup
  • A laptop with Node.js installed;
  • An Android or iOS device.
Required experience

Basic to intermediate Angular knowledge.

Our target for the day

We will convert a more complex Angular app with lazy modules, guards and custom web and mobile plugins.
