Moving to Angular with ngUpgrade

Thursday 14th March, 10.00-14.00
Solving business problems

If you've got a medium-sized AngularJS application that's too big to rewrite from scratch but too small to need a complex strategy like hybrid routing, you're a great candidate for the ngUpgrade library.

Getting started can feel really overwhelming, though. Where to begin? What should you most focus your time on? How do you migrate components, services, and other AngularJS bits over to Angular?

Sam Julien will walk you through all of these concerns in this step-by-step interactive workshop on ngUpgrade. You'll learn how to prepare for and plan your migration, how to set up ngUpgrade so that both Angular and AngularJS can run side-by-side, and how to use ngUpgrade to migrate your components and services. You'll also learn some practical techniques for build tooling and production compilation.

This workshop will be very interactive with lots of exercises and discussion. We'll also make sure you walk away with plenty of sample code and supplemental materials so you can solidify what you've learned when you get back to work.

Trainer

Sam Julien, creator of upgradingangularjs.com, Google Developer Expert

NgRx - Mastering Advanced State Management

Friday 15th March, 10.00-17.00
Be the expert

Do you already have good knowledge of Angular and Typescript? Some hands-on experience with NgRx? And are you already familiar with the basics of ngrx/store?

As your application grows, at some point you will face several new challenges where you will not find a lot of information about best practices or ready solutions.

In such situations we often tend to end up in poor implementations or hacky workarounds.

Be different! Develop a critical understanding for state management. Learn several strategies to tackle advanced scenarios and benefit from knowing when to use the different approaches.

In a full-day in-depth course on NgRx you will learn advanced approaches such as facades, optimistic updates, memoized selector composition, and many other tricks.

At the end you will know everything you need to know about:

  • Working with feature stores
  • Using @ngrx/entities
  • Handling state in lazy loaded application parts
  • Implementing the facade pattern
  • Using effects to run optimistic updates
  • Selector composition and memoized selectors
  • Implementing immutability through meta reducers
